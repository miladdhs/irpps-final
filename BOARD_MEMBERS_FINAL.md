## ویژگی‌های پیاده‌سازی

1. **نمایش به تفکیک دوره**: هر دوره در یک بخش جداگانه نمایش داده می‌شود
2. **طراحی ریسپانسیو**: در موبایل 2 ستون، تبلت 3 ستون، دسکتاپ 4 ستون
3. **آیکون پیش‌فرض**: برای هر عضو یک آیکون person نمایش داده می‌شود
4. **اطلاعات نمایش داده شده**:
   - نام کامل
   - سمت
   - تخصص

## نکات

- اطلاعات به صورت استاتیک (هاردکد) در کد قرار دارند
- عکس‌های پروفایل در حال حاضر وجود ندارند (آیکون پیش‌فرض نمایش داده می‌شود)
- در صورت نیاز به تغییر اطلاعات، باید فایل `About.vue` را ویرایش کنید

## آماده برای Deploy ✅

فایل آماده است و می‌توانید آن را روی سرور آپلود کنید.

```bash
# با git
git add frontend/src/views/About.vue
git commit -m "Add board members information to About page"
git push

# یا با scp
scp frontend/src/views/About.vue user@server:/path/to/frontend/src/views/
```

بعد از آپلود، frontend را rebuild کنید:
```bash
cd frontend
npm run build
```
