# راهنمای راه‌اندازی اطلاعات هیئت مدیره

## مشکل
اطلاعات هیئت مدیره انجمن در صفحه "درباره ما" به صورت هاردکد بود و از دیتابیس لود نمی‌شد.

## راه‌حل

### 1. اضافه کردن داده‌های هیئت مدیره به دیتابیس

یک اسکریپت Python ایجاد شده است که اطلاعات تمام اعضای هیئت مدیره را در سه دوره (1395، 1400، 1403) به دیتابیس اضافه می‌کند:

```bash
# اجرا در محیط Docker
cd backend
bash run_add_board_members.sh

# یا اجرا مستقیم در کانتینر
docker-compose exec backend python add_board_members.py
```

### 2. API Endpoint جدید

یک endpoint جدید به API اضافه شده است:

```
GET /api/accounts/board-members/
```

این endpoint اطلاعات اعضای هیئت مدیره را به تفکیک دوره برمی‌گرداند:


### 3. تغییرات در Frontend

صفحه `About.vue` باید به‌روزرسانی شود تا داده‌ها را از API دریافت کند به جای استفاده از داده‌های هاردکد.

#### تغییرات مورد نیاز:

1. اضافه کردن state برای نگهداری داده‌های هیئت مدیره
2. ایجاد تابع برای fetch کردن داده‌ها از API
3. نمایش داده‌ها به تفکیک دوره

## نکات مهم

1. اسکریپت `add_board_members.py` باید فقط یک بار اجرا شود
2. اگر نیاز به به‌روزرسانی اطلاعات باشد، می‌توانید اسکریپت را دوباره اجرا کنید (اطلاعات موجود به‌روزرسانی می‌شوند)
3. تمام اعضای هیئت مدیره با username به فرمت `board_نام_خانوادگی` ایجاد می‌شوند
4. وضعیت تایید (approval_status) تمام اعضا به صورت خودکار روی 'approved' تنظیم می‌شود

## تست API

برای تست API می‌توانید از curl استفاده کنید:

```bash
# تست endpoint هیئت مدیره
curl http://localhost:8000/api/accounts/board-members/

# یا از طریق nginx
curl http://localhost/api/accounts/board-members/
```
